# Logo para PDFs

Para que el logo aparezca en los PDFs (resúmenes de pago y presupuestos), coloca tu archivo PNG del logo aquí con el nombre:

**`logo.png`**

El archivo debe estar en esta carpeta (`public/logo.png`) para que funcione correctamente.

## Ubicación del archivo:
```
mini-crm-next/
  └─ public/
      └─ logo.png  ← Coloca tu logo aquí
```

## Notas:
- El logo debe ser en formato PNG
- Se mostrará en el footer de todos los PDFs generados
- Si el logo no se encuentra, se usará el texto "Digital Space" como fallback
